VideoPro-Social Accounts

Use this widget to show social accounts on the sidebar.

Parameters:

  • Title: Title of widget.
  • Widget Logic: Enter widget logic here.
  • Title Icon Class: Enter Font Awesome Class for the icon.
  • Badge Text: Enter badge text.
  • Badge Text Color: Choose text color for the badge.
  • Badge Background Color: Choose background color for the badge.
  • Widget Styles: Choose widget styles from drop down list styles.
  • Widget Custom Variation: Enter your Custom CSS class.
